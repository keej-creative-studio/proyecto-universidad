# Proyecto de final de cuatrimestre

# Requerimientos
- Godot 4.6.1 o Mayor

# Estructura del proyecto:

**/addons:** En esta carpeta van todos los complementos que instalamos al proyecto.

- characters: assets de personajes
- ui: assets de la interfaz de usuario
- world: assets del nivel

**/assets:** En esta carpeta van todos los modelos 3d, texuras e imagenes y sonidos que usan el proyecto.

**/scenes:** En esta carpeta van todas las escenas de de godot del videojuego.

- menus: escenas relacionadas a los controles de menu.
- network: escenas relacionadas a controles de red
- player: escanas relacionadas a los jugadores
- world: escenas relacionadas a los niveles

**/scripts:** En esta carpeta van todos los scripts de codigo del videojuego, el lenguaje usado es gdscript.
